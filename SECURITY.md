# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Responsible Use

CyberShield Toolkit is designed for **authorized security testing and educational purposes only**. Users must:

- Only scan networks, domains, and systems they own or have explicit written permission to test.
- Comply with all applicable local, state, and federal laws.
- Not use this tool for unauthorized access, disruption, or any malicious activity.

The author assumes no liability for misuse of this software.

## Reporting a Vulnerability

If you discover a security vulnerability in the toolkit itself, please report it responsibly:

1. **Do not** open a public GitHub issue.
2. Email the maintainer directly or use GitHub's private vulnerability reporting feature.
3. Include a description of the vulnerability, steps to reproduce, and potential impact.

I aim to acknowledge reports within 48 hours and provide a fix or mitigation within 7 days for confirmed issues.
